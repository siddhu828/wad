WHAT IS jQuery?

jQuery is a JavaScript library.

It simplifies:

* JavaScript coding
* DOM manipulation
* animations
* event handling

⸻

WHAT IS jQuery MOBILE?

jQuery Mobile is a framework built using:

* jQuery
* HTML
* CSS

Used for:

* mobile-friendly websites
* responsive web applications
* touch-based UI

⸻

WHAT IS A MOBILE WEBSITE?

A mobile website is:

* optimized for small screens
* touch-friendly
* responsive

⸻

WHAT IS RESPONSIVE DESIGN?

Responsive design automatically adjusts:

* layout
* buttons
* spacing
* UI

according to screen size.

⸻

WHAT IS CDN?

CDN stands for: Content Delivery Network
- it loads libararis directly from the internet

WHY WE USED CDN LINKS

Advantages:

* no installation required
* easier setup
* faster development
* lightweight project

IMPORTANT jQuery MOBILE ATTRIBUTES

data-role="page"

Creates a mobile page.
eg: <div data-role="page">

data-role="header"

Creates mobile header/navbar.

Example: <div data-role="header">

WHAT IS href="#about" ?

Navigates INSIDE same HTML file.

Example:<a href="#about"> moves to <div id="about">

What is data-role?

Special jQuery Mobile attribute used to define UI components.

⸻

Why use viewport meta tag?

Makes website responsive on mobile devices.
