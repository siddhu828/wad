⸻

WHAT IS NODE.JS?

Node.js is a JavaScript runtime environment.

Normally:

JavaScript runs only inside browser.

With Node.js:

JavaScript can run outside browser on server.

Node.js is used for:

* backend development
* APIs
* servers
* database operations
* real-time applications

⸻

WHAT IS EXPRESS.JS?

Express.js is a framework built on top of Node.js.

It simplifies:

* routing
* server creation
* APIs
* middleware handling

Without Express:
Node.js backend becomes more complicated.

⸻

WHAT IS A STATIC WEBSITE?

A static website contains files that do not change dynamically.

Examples:

* HTML
* CSS
* images

Static websites do not use database processing.

STEP 5 — WRITE SERVER CODE

Open:

server.js

Write:

const express = require('express');
const path = require('path');
const app = express();
const PORT = 3000;
app.use(express.static(path.join(__dirname, 'public')));
app.listen(PORT, () => {
    console.log(`Server running on port ${PORT}`);
});

⸻

EXPLANATION OF EACH LINE

Import Express

const express = require('express');

Imports Express library.

⸻

Import Path Module

const path = require('path');

Used for:

* file paths
* folder paths

Built-in Node.js module.

⸻

Create Express App

const app = express();

Creates Express application/server.

⸻

Define Port

const PORT = 3000;

Application runs on:

localhost:3000

⸻

Serve Static Files

app.use(express.static(path.join(__dirname, 'public')));

Very important line.

Explanation:

Part	Meaning
app.use()	Uses middleware
express.static()	Serves static files
__dirname	Current project directory
public	Static folder

This allows browser to access:

* HTML
* CSS
* images

from public folder.

⸻

Start Server

app.listen(PORT, () => {
    console.log(`Server running on port ${PORT}`);
});

Starts Node.js server.

⸻

WHAT IS app.listen()?

It tells server to:

* start running
* listen for browser requests

STEP 6 — CREATE HOME PAGE

Open:

public/index.html

Write:

<!DOCTYPE html>
<html>
<head>
    <title>Home</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>Welcome to Node.js Website</h1>
        <p>This is the Home Page</p>
        <a href="about.html">About</a>
        <a href="contact.html">Contact</a>
    </div>
</body>
</html>

⸻

EXPLANATION

Tag	Purpose
DOCTYPE	HTML5 declaration
head	Metadata section
title	Browser tab title
link	Connect CSS file
body	Main content
a	Hyperlink
